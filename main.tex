\begin{block}{Abstract}

Eigenvalues are ubiquitous in all branches of science and engineering. The dominant eigenvalue algorithm reduces input matrices to almost diagonal and then creates and chases bulges using parameters called shifts. Good parameter choices can allow shrinking of the original problem allowing for a more aggressive deflation on the subproblem. Such deflations greatly improve performance. We apply automatic differentiation tools to achieve frequent deflations.

\end{block}

\begin{alertblock}{Motivation}

We wish to utilize autodifferentation to help optimize the process of finding eigenvalues for deflation
\begin{itemize}
\item Use QR algorithm and chase the resulting buldge to the bottom.
\item Use shure decomposition to obtain a spike with eigenvalue information.
\item Increase the frequency of these deflations by using automatic differentiation tools to inspire shift choices.
\item Eventually alter algorithm to a middle deflation for greatly reduced problem size.
\end{itemize}

\end{alertblock}

\begin{block}{Methods}
Taking our matrix we first start by changing it into a Hessenberg form. Next the implicit QR algorithim brings the buldge to the bottom. We then inflate the bludge using Schure decomposition. Finally we look at the sensitivity of the spikes from our shift. This is done using automatic differentiation tools. This will alow for a better choice in a shift which will improve performance and possibly result in a smaller subproblem.

\end{block}
